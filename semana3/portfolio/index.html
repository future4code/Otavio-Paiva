Me chamo Otávio Augusto, sou Front End Developer Jr.

Atuo como Desenvolvedor Front-End Junior, que busca soluções para problemas reais do dia-a-dia através da computação.
Busco atuar em projetos que visam impactar a sociedade em geral, com objetivo de apresentar soluções aos problemas enfrentados pelos usuários, através de experiências reais de programação.

Gosto de criar interfaces seguindo conceitos modernos, responsivos e clean, sempre com foco em UI/UX. Minhas stacks são apresentáveis em Html, CSS, JavaScript, React, Typescript e Git, com CSS utilizo bibliotecas como Styled-Components, SASS e Chakra UI.
Sigo em constante aprendizado, busco aprender Banco de dados(SQL), back-end utilizando Node e mobile utilizando React Native.

Atualmente estou fazendo o curso Full-Stack da Labenu, que vem melhorando muito meus resultados, principalmente me possibilitando a aprender realizar atividades em grupo, executar tarefas que são reais no cotidiano das empresas, como o uso do Git para versionamento dos códigos.
